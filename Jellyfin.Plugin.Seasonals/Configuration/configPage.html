Main Plugin Settings
Enable or disable the entire plugin functionality.
If enabled, the plugin will use the rules defined in the "Auto Selection" tab to assume the season. If no rule matches, it falls back to the "Standard Season" below.
If enabled, users will see a seasonals icon in the header to toggle seasonals for their browser (device-specific).
If enabled, Seasonals effects will be disabled by default on TV devices. Users can still manually override this in their client settings.
The season to display if automation is disabled or no "Auto Selection" rule matches the current date.
Choose where the client-side settings menu can be accessed.
Choose where the client-side settings menu can be accessed on mobile devices.
Auto Selection Rules
Define rules to automatically select a season based on the date. Rules are evaluated from top to bottom. The first matching rule wins.
